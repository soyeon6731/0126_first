## 1. 사이트맵

### 1.1 전체 구조 (단일 페이지)

```
KURATION
│
└── 챗봇 홈 (Single Page Application)
    ├── 헤더 (로고)
    ├── 챗봇 대화 영역
    │   ├── 메시지 스레드
    │   └── 추천 결과 카드
    └── 입력 영역
        ├── 텍스트 입력창
        └── 전송 버튼
```

**페이지 수**: 1개 (챗봇 홈 페이지만)

## 4. 컴포넌트 계층 구조

### 4.1 전체 컴포넌트 트리

```
App
├── Header
│   └── Logo
│
└── ChatPage
    ├── ChatArea
    │   └── MessageThread
    │       ├── BotMessage (웰컴)
    │       ├── UserMessage
    │       ├── BotMessage (추천 결과)
    │       │   └── RecommendationCard
    │       │       └── SongItem (반복)
    │       │           ├── SongInfo (제목, 아티스트)
    │       │           └── PlatformLinks
    │       │               ├── YouTubeButton
    │       │               └── SpotifyButton
    │       └── LoadingIndicator
    │
    └── InputArea
        ├── TextInput
        └── SendButton
```

### 4.2 주요 컴포넌트 상세

#### 4.2.1 ChatArea
```jsx
<ChatArea>
  <MessageThread>
    {messages.map(msg => (
      msg.type === 'user' ?
        <UserMessage text={msg.text} /> :
        <BotMessage
          text={msg.text}
          recommendations={msg.recommendations}
        />
    ))}
  </MessageThread>
</ChatArea>
```

#### 4.2.2 RecommendationCard
```jsx
<RecommendationCard>
  {songs.map((song, idx) => (
    <SongItem key={song.id}>
      <span>{idx + 1}. {song.title} - {song.artist}</span>
      <PlatformLinks>
        <YouTubeButton url={song.youtubeUrl} />
        <SpotifyButton url={song.spotifyUrl} />
      </PlatformLinks>
    </SongItem>
  ))}
</RecommendationCard>
```

## 5. 인터랙션 패턴

### 5.1 메시지 전송

**데스크톱:**
```
1. 사용자가 텍스트 입력
2. Enter 키 또는 전송 버튼 클릭
3. 입력창 비활성화
4. 로딩 스피너 표시
5. LLM 응답 대기
6. 결과 카드 표시
7. 입력창 재활성화
```

**모바일:**
```
1. 입력 필드 탭
2. 소프트 키보드 표시
3. 메시지 입력
4. 전송 버튼 탭
5. 키보드 숨김
6. 로딩 표시
7. 결과 카드 슬라이드 업
```

### 5.2 플랫폼 링크 클릭

```
1. YouTube/Spotify 버튼 클릭
2. 새 탭으로 외부 링크 열림
3. 원래 페이지는 유지
```

## 6. 반응형 디자인

### 6.1 브레이크포인트

| 디바이스 | 해상도 | 레이아웃 변경사항 |
|---------|--------|------------------|
| 모바일 | < 768px | 전체 화면 대화창, 카드 1열 |
| 태블릿/데스크톱 | ≥ 768px | 중앙 정렬, 최대 너비 800px |

### 6.2 모바일 최적화

```
- 입력창 하단 고정
- 터치 버튼 최소 44x44px
- 가로 스크롤 방지
- 대화창 자동 스크롤 (최신 메시지)
```
